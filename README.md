# Student Performance Dashboard
An interactive Web Dashboard built with Streamlit to analyze student exam performance.

## Features
- Interactive Filtering by Pass/Fail status.
- Automated Statistical Summary.
- Visual Analytics (Count Plots and Heatmaps).

## How to Run:
1. Install requirements: pip install -r requirements.txt
2. Launch the app: streamlit run app.py or visit https://student-performance-dashboard-akp5rswzsm3km6tlw573pf.streamlit.app/
